## ❌ 失败页面（必须修复）

### 1. /semi — 半导体主题
- **错误**: `Minified React error #310` — hooks 调用顺序不一致
- **组件**: SemiTheme
- **表现**: ErrorBoundary 触发, 页面显示"页面渲染异常"
- **修复方向**: 检查条件渲染中的 hooks 调用 (if/return 前必须保证 hooks 数量一致)

### 2. /qmt — QMT 实盘
- **错误**: `Cannot read properties of undefined (reading 'toLocaleString')`
- **组件**: QMTSpot
- **表现**: ErrorBoundary 触发, 页面显示"页面渲染异常"
- **修复方向**: 某个数值格式化前为 undefined, 需做空值保护

### 3. /tasks — 任务中心
- **错误**: `rawData.some is not a function`
- **组件**: TaskCenter (InternalTable)
- **表现**: ErrorBoundary 触发, 页面显示"页面渲染异常"
- **修复方向**: API 返回非数组数据, antd Table dataSource 需要 `Array.isArray` 检查
